# HubEx design system
- Components: install `@cvetkov_es/react` + `@cvetkov_es/css`; `import "@cvetkov_es/css"` once at app entry; import components from `@cvetkov_es/react` before hand-rolling UI.
- Rules & tokens: read the design reference at `https://raw.githubusercontent.com/cvetkov-es/HubEx.Design/master/llms.txt` — it is derived from the official `@hubex/design-system@0.4.0` (themes.HubEx), not from Figma; treat it, not any cached Figma export, as the ground truth for token values and component props.
